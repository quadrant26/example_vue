vue-cli  vue 脚手架

    提供剧本的项目结构

项目模板

    simple
    webpack
    webpack-simple

    browserify
    browserify-simple

use

1. npm install vue-cli -g           安装环境

    验证 vue --version

2. vue init <模板名称 simple | webpack | webpack-simple> 本地文件夹(folder)

    vue init simple#1.0 filefolder

3. 进入目录

    npm install

4. npm install dev
